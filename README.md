# 基于bootstrap5的响应式静态界面

#### 介绍
这是一个基于bootstrap5的响应式静态界面，目前已经基本可以自适应，但是还有一些不足，欢迎fork优化哦~

#### 软件架构
主要是使用了bootstrap5进行响应式开发，以及使用了jquery进行轮播图的图片切换以及请求第三方接口更换图片~
